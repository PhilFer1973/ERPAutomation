# V2 Roadmap

Future phases may add:

- Approval workflow before supplier creation
- Bank detail capture with stronger controls
- Duplicate supplier detection
- Finance/admin failure notifications
- Retry logic for recoverable failures
- SharePoint document library storage for screenshots
- Dashboard for queue and run status
- Supplier amendment process
- Supplier deletion controls
- AP invoice entry prototype
- Stronger credential storage
- Unattended automation, if licensed and appropriate
- Human-in-the-loop review screen
- Teams notifications
- Replay mode for saved screenshots
- Prompt evaluation with known screenshot examples

Any expansion should preserve the V1 control principle: PAD executes the known route, and Vision validates checkpoints.
